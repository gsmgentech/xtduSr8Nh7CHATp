Before You Begin

  • Ensure you have a stable internet connection.
  • Prepare the USB device and appropriate data cable.
  • Use a Windows-based computer or laptop.

Connection Steps

  1. Download and install USB Redirector (Customer Module).
  2. Launch the application using Run as Administrator.
  3. In the Connect With Technician window:
    • Enter the Technician ID provided by our support team.
    • Enter your name in the Additional Information field.
    • Click Connect to start the session.
  4. Connect the USB device you wish to share. If already connected, unplug and reconnect it to ensure detection.
  5. Wait while the technician performs the required service.

Security Notice: Only the selected USB device is accessible during the session. No other files, applications, or personal data on your computer are exposed.

Troubleshooting

  • Unable to Connect: Verify the Technician ID and check your internet connection.
  • USB Device Not Detected: Restart the application as Administrator and reconnect the device.
  • Security Warning: Allow the application only if downloaded from the official source.
